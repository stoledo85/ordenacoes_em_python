# Projeto com as famosas ordenações em Python

Projeto estará testando as mais famosas ordenações de números com listas de 25 termos até 1000. Será testado o tempo de execução de cada um.

## Quais serão os metodos escolhidos

* [Bubble]('https://pt.wikipedia.org/wiki/Bubble_sort')
* [Selection]('https://pt.wikipedia.org/wiki/Selection_sort')
* [Insertion]('https://pt.wikipedia.org/wiki/Insertion_sort')
* [Comb]('https://pt.wikipedia.org/wiki/Comb_sort')
* [Bogo]('https://pt.wikipedia.org/wiki/BogoBusca')
* [Merge]('https://pt.wikipedia.org/wiki/Merge_sort')
* [Heap]('https://pt.wikipedia.org/wiki/Heapsort')
* [Gnome]('https://pt.wikipedia.org/wiki/Gnome_sort')
* [Bucket]('https://pt.wikipedia.org/wiki/Bucket_sort')
* [Cocktail]('https://pt.wikipedia.org/wiki/Cocktail_sort')
* [Quick]('https://pt.wikipedia.org/wiki/Quick_sort')
* [Shell]('https://pt.wikipedia.org/wiki/Shell_sort')
